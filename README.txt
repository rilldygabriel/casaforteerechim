CASA FORTE SITE V1

Arquivos principais:
- index.html
- styles.css
- script.js
- pasta assets (logos e fotos originais, sem alteração)

Para publicar no GitHub:
1. Descompacte este ZIP.
2. No repositório casaforteerechim, clique Add file > Upload files.
3. Selecione TODOS os arquivos e a pasta assets.
4. Clique Commit changes.
5. A Vercel fará a publicação automaticamente.

Observação: nesta V1, os formulários abrem o WhatsApp com os dados preenchidos. Eles ainda não armazenam respostas em banco de dados.
